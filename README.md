# Finance Tracker 

A Python-based personal finance tracking system.

## Features
- Add income and expenses
- Categorize transactions
- View all records
- Delete transactions
- Calculate balance
- Save data to file (JSON)

## Tech Used
- Python
- File handling (JSON)

## How to Run
1. Clone repo
2. Run:
   python finance_tracker.py
